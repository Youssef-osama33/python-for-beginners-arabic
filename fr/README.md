# 🐍 Python pour Débutants (Français)

Bienvenue dans le parcours français de l'Ultimate Global Python Repository ! Ce guide est conçu pour vous emmener de zéro à héros dans la programmation Python.

## 🚀 Pourquoi Python ?
Python est l'un des langages de programmation les plus populaires au monde car :
- **Facile à apprendre :** Sa syntaxe est proche de l'anglais.
- **Polyvalent :** Utilisé dans le développement Web, l'IA, la science des données et l'automatisation.
- **Communauté immense :** Des millions de développeurs et de bibliothèques prêtes à l'emploi.

## 🛠️ Comment Commencer ?
1. Installez Python depuis le site officiel : [python.org](https://www.python.org/).
2. Clonez ce dépôt :
   ```bash
   git clone https://github.com/Youssef-osama33/python-for-beginners-arabic.git
   ```
3. Accédez au dossier `fr/` et commencez par la première leçon.
