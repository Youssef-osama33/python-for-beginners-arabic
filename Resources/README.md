# Resources (المصادر والمراجع) 📚

فيما يلي مجموعة من أفضل المصادر لتعلم لغة بايثون باللغة العربية والإنجليزية:

## 🇸🇦 مصادر باللغة العربية
- **[قناة بايثون بالعربي على يوتيوب](https://www.youtube.com/@PythonArabi):** دروس شاملة ومبسطة للمبتدئين.
- **[موقع هرمش (Harmash)](https://harmash.com/tutorials/python):** مرجع ممتاز لتعلم البرمجة بلغة بايثون من الصفر.
- **[أكاديمية حسوب](https://academy.hsoub.com/programming/python/):** مقالات ودورات متخصصة في بايثون.
- **[كتاب "ألف باء بايثون"](https://github.com/nalomran/Learn-Python-In-Arabic):** كتاب مجاني ومفتوح المصدر لتعلم بايثون.

## 🇺🇸 مصادر باللغة الإنجليزية
- **[Python.org Documentation](https://docs.python.org/3/tutorial/):** التوثيق الرسمي للغة بايثون.
- **[FreeCodeCamp (Python for Beginners)](https://www.youtube.com/watch?v=rfscVS0vtbw):** كورس فيديو شامل مدته 4 ساعات.
- **[Real Python](https://realpython.com/):** مقالات ودروس متقدمة في بايثون.
- **[W3Schools Python Tutorial](https://www.w3schools.com/python/):** دروس تفاعلية وسهلة للمبتدئين.

## 🛠️ أدوات مفيدة
- **[Visual Studio Code](https://code.visualstudio.com/):** أفضل محرر أكواد لبايثون.
- **[PyCharm](https://www.jetbrains.com/pycharm/):** بيئة تطوير متكاملة (IDE) قوية جداً.
- **[Jupyter Notebook](https://jupyter.org/):** أداة رائعة لتحليل البيانات والتعلم التفاعلي.
