## مقدمة

السلاسل النصية (Strings) هي نصوص محاطة بعلامات اقتباس سواء مفردة `'...'` أو مزدوجة `"..."`. تحتوي بايثون على العديد من الطرق (Methods) التي يمكن استخدامها مع السلاسل النصية لتعديلها، فحص محتواها، أو تحويلها.

في الدروس السابقة تعلمنا العديد من الطرق مثل:
- `upper()`, `lower()`
- `strip()`, `replace()`
- `split()`, `join()`

في هذا الدرس سنستعرض **مجموعة جديدة من الطرق** المهمة.

## 1. طريقة `startswith()`

- تستخدم لفحص إذا كان النص يبدأ بسلسلة معينة.
- تعيد `True` إذا كان النص يبدأ بالجزء المحدد، و`False` خلاف ذلك.

### الصيغة:
```python
string.startswith(substring)
```

### مثال:

```python
text = "مرحبًا بك في بايثون"
print(text.startswith("مرحبًا"))  # True
print(text.startswith("بايثون"))  # False
```

---

## 2. طريقة `endswith()`

- تستخدم لفحص إذا كان النص ينتهي بسلسلة معينة.
- تعيد `True` إذا كان النص ينتهي بالجزء المحدد، و`False` خلاف ذلك.

### الصيغة:
```python
string.endswith(substring)
```

### مثال:

```python
text = "تعلم البرمجة مع بايثون"
print(text.endswith("بايثون"))  # True
print(text.endswith("جافا"))    # False
```

---

## 3. طريقة `count()`

- تستخدم لحساب عدد مرات ظهور نص معين داخل السلسلة.

### الصيغة:
```python
string.count(substring)
```

### مثال:

```python
text = "بايثون لغة برمجة رائعة، بايثون سهلة التعلم"
print(text.count("بايثون"))  # 2
print(text.count("لغة"))     # 1
```

---

## 4. طريقة `find()`

- تبحث عن أول ظهور لنص معين في السلسلة.
- تعيد موقع أول حرف من النص داخل السلسلة.
- إذا لم تجد النص تعيد القيمة `-1`.

### الصيغة:
```python
string.find(substring)
```

### مثال:

```python
text = "مرحبا بكم في عالم بايثون"
print(text.find("عالم"))  # 13 (موقع أول حرف من "عالم")
print(text.find("جافا"))  # -1 (غير موجود)
```

---

## 5. طريقة `rfind()`

- مثل `find()` ولكن تبحث من نهاية السلسلة (البحث من اليمين إلى اليسار).

### الصيغة:
```python
string.rfind(substring)
```

### مثال:

```python
text = "بايثون لغة برمجة بايثون"
print(text.rfind("بايثون"))  # 16 (موقع آخر ظهور)
```

---

## 6. طريقة `capitalize()`

- تحول أول حرف في النص إلى حرف كبير (Uppercase) والباقي إلى صغير (Lowercase).

### الصيغة:
```python
string.capitalize()
```

### مثال:

```python
text = "بايثون رائعة"
print(text.capitalize())  # "بايثون رائعة" (في حالة نص عربي الحروف تبقى كما هي)
text2 = "python language"
print(text2.capitalize())  # "Python language"
```

---

## 7. طريقة `title()`

- تحول أول حرف من كل كلمة إلى حرف كبير.

### الصيغة:
```python
string.title()
```

### مثال:

```python
text = "تعلم البرمجة مع بايثون"
print(text.title())  # "تعلم البرمجة مع بايثون"
text2 = "python language"
print(text2.title())  # "Python Language"
```
