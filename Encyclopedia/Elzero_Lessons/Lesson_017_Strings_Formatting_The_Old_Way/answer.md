# حل تحدي الدرس 17: Strings Formatting The Old Way

## ✅ الحل

بالطبع! سأقدم لك حلاً كاملاً ومفصلاً مع شرح خطوة بخطوة لكيفية استخدام طريقة تنسيق النصوص القديمة % في بايثون لحل هذا التحدي.

---

## السؤال:
اكتب دالة باسم `format_product_info` تستخدم طريقة التنسيق القديمة `%` لتنسيق وعرض معلومات منتج بحيث:
- تأخذ 3 معطيات: اسم المنتج (نص)، السعر (float)، والكمية (int).
- تعيد نصاً بالشكل:
  
  ```
  "تم شراء المنتج '%s' بسعر %0.2f دولار, والكمية المتوفرة هي %d قطع."
  ```

---

## الحل:

### 1. كتابة الدالة

```python
def format_product_info(product_name, price, quantity):
    # نستخدم التنسيق القديم % مع السلسلة النصية
    formatted_string = "تم شراء المنتج '%s' بسعر %0.2f دولار, والكمية المتوفرة هي %d قطع." % (product_name, price, quantity)
    return formatted_string
```

---

### 2. شرح الكود خطوة بخطوة:

- `def format_product_info(product_name, price, quantity):`
  - هذا تعريف الدالة التي تأخذ 3 معطيات:
    - `product_name` (اسم المنتج نصي)
    - `price` (السعر عدد عشري float)
    - `quantity` (الكمية عدد صحيح int)
  
- `"تم شراء المنتج '%s' بسعر %0.2f دولار, والكمية المتوفرة هي %d قطع."`
  - هذا النص الثابت الذي نريد عرضه مع التنسيق.
  - `%s` → مكان لتنسيق نص (اسم المنتج)
  - `%0.2f` → مكان لتنسيق عدد عشري float، مع عرض دقيقتين عشريتين دائماً.
  - `%d` → مكان لتنسيق عدد صحيح int (الكمية)
  
- `% (product_name, price, quantity)`
  - يتم استبدال علامات التنسيق بالقيم الموجودة في المتغيرات بالترتيب:
    - `%s` يستبدل بـ `product_name`
    - `%0.2f` يستبدل بـ `price`
    - `%d` يستبدل بـ `quantity`
  
- `return formatted_string`
  - تعيد النص النهائي بعد التنسيق.

---

### 3. اختبار الدالة مع مثال:

```python
# اختبار الدالة
result = format_product_info("كمبيوتر", 399.99, 5)
print(result)
```

### الناتج سيكون:

```
تم شراء المنتج 'كمبيوتر' بسعر 399.99 دولار, والكمية المتوفرة هي 5 قطع.
```

---

### 4. توضيحات إضافية:

- `%s` يُستخدم لتنسيق النصوص.
- `%0.2f` يعني:
  - `f` لتنسيق الرقم كرقم عشري float.
  - `.2` تعني عرض رقمين بعد الفاصلة العشرية.
  - `0` (اختياري) يعني عرض الرقم بدون ترك مساحة فارغة.
- `%d` لتنسيق الأعداد الصحيحة.

---

## ملخص

```python
def format_product_info(product_name, price, quantity):

print(format_product_info("كمبيوتر", 399.99, 5))
```
