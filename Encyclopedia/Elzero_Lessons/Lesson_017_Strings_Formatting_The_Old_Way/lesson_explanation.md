# درس بايثون رقم 17: تنسيق النصوص بالطريقة القديمة (Strings Formatting The Old Way)

## مقدمة
في بايثون، غالبًا ما نحتاج إلى دمج المتغيرات داخل النصوص (Strings) بطريقة منظمة وسلسة. واحدة من الطرق المعروفة لتنسيق النصوص هي **الطريقة القديمة** التي تستخدم عامل `%` (Percentage operator).

هذه الطريقة قديمة لكنها بسيطة ومفيدة لفهم أساسيات تنسيق النصوص، وتعمل في إصدارات بايثون القديمة والجديدة.

---

## المفهوم الأساسي

عندما نكتب نصًا ونريد إدخال قيم متغيرة بداخله، نستخدم رموز خاصة (Placeholders) داخل النص تمثل مكان المتغيرات التي نريد إدخالها.

### الرموز المستخدمة في الطريقة القديمة:

| الرمز | المعنى                             | مثال على القيمة                  |
|-------|-----------------------------------|--------------------------------|
| `%s`  | نص (String)                      | `"مرحبا"`                      |
| `%d`  | عدد صحيح (Integer)                | `10`                           |
| `%f`  | عدد عشري (Float)                  | `3.1415`                       |
| `%.2f`| عدد عشري مع تحديد عدد الأرقام بعد الفاصلة | `3.14` (بعد تقريب لأثنين خانة) |

---

## كيف نستخدم هذه الرموز؟

نضع النص الذي يحتوي على الرموز، ثم نستخدم `%` وبعده نضع المتغيرات التي نريد إدخالها، إما بشكل فردي أو كمجموعة (Tuple).

---

## أمثلة تطبيقية

### مثال 1: نص مع متغير نصي فقط

```python
name = "أحمد"
print("مرحباً %s!" % name)
```

**النتيجة:**

```
مرحباً أحمد!
```

---

### مثال 2: نص مع متغير عددي (عدد صحيح)

```python
age = 25
print("عمري %d سنة." % age)
```

**النتيجة:**

```
عمري 25 سنة.
```

---

### مثال 3: نص مع عدد عشري

```python
pi = 3.14159265
print("قيمة باي تقريبًا: %f" % pi)
```

**النتيجة:**

```
قيمة باي تقريبًا: 3.141593
```

> لاحظ أن `%f` يعرض 6 أرقام بعد الفاصلة بشكل افتراضي.

---

### مثال 4: تحديد عدد الأرقام بعد الفاصلة العشرية

```python
pi = 3.14159265
print("باي بدقة منزلتين عشريتين: %.2f" % pi)
```

**النتيجة:**

```
باي بدقة منزلتين عشريتين: 3.14
```

---

### مثال 5: تنسيق نص مع أكثر من متغير

```python
name = "سارة"
age = 30
print("الاسم: %s، العمر: %d سنة." % (name, age))
```

**النتيجة:**

```
الاسم: سارة، العمر: 30 سنة.
```

> لاحظ أننا وضعنا المتغيرات داخل قوسين `()` كـ Tuple بعد علامة `%`.

---

## ملاحظات مهمة

- يجب أن يتوافق نوع المتغير مع رمز التنسيق المستخدم (`%s` للنصوص، `%d` للأعداد الصحيحة، `%f` للأعداد العشرية).
- يمكن استخدام `%s` لأي نوع، لأنه يقوم بتحويل القيمة إلى نص تلقائياً.
- هذه الطريقة قديمة، ويوجد بدائل أحدث مثل `.format()` و f-strings، لكنها مفيدة لفهم أساسيات التنسيق.

---

## ملخص سريع

| خطوة                                     | مثال                           |
|-----------------------------------------|-------------------------------|
| اكتب النص مع رموز التنسيق               | `"مرحباً %s!"`                |
| استخدم `%` مع المتغيرات                 | `% name`                      |
| للنصوص المتعددة، اجمعها في Tuple        | `% (name, age)`               |
| استخدم الرموز المناسبة (`%s`, `%d`, `%f`) | `%s` للنص، `%d` للأعداد، `%f` للعشرية |

---

## أسئلة تحدي

1. اكتب برنامجًا يطبع جملة تحتوي على اسم منتج وسعره (عدد عشري بدقة منزلتين).
2. كيف يمكن طباعة عدد صحيح داخل نص باستخدام الطريقة القديمة؟
3. ماذا يحدث إذا استخدمت `%d` مع قيمة نصية؟ جرب ذلك وفسر النتيجة.
