---

## 1. ما هي الأعداد في بايثون؟

الأعداد هي نوع بيانات يُستخدم لتمثيل القيم العددية. في بايثون، هناك عدة أنواع من الأعداد، أشهرها:

- **الأعداد الصحيحة (Integers)**: وهي الأعداد بدون أجزاء عشرية، مثل 5، -10، 0.
- **الأعداد العشرية أو الفاصلة العائمة (Floating point numbers)**: وهي الأعداد التي تحتوي على جزء عشري، مثل 3.14، -0.5.
- **الأعداد المركبة (Complex numbers)**: وهي أعداد تحتوي على جزء حقيقي وجزء تخيلي، مثل 3+4j.

---

## 2. تعريف الأعداد في بايثون

### الأعداد الصحيحة (int)

```python
a = 10
b = -5
print(type(a))  # <class 'int'>
print(type(b))  # <class 'int'>
```

### الأعداد العشرية (float)

```python
x = 3.14
y = -0.001
print(type(x))  # <class 'float'>
print(type(y))  # <class 'float'>
```

### الأعداد المركبة (complex)

```python
z = 3 + 4j
print(type(z))  # <class 'complex'>
print(z.real)   # الجزء الحقيقي: 3.0
print(z.imag)   # الجزء التخيلي: 4.0
```

---

## 3. العمليات الحسابية على الأعداد

يمكنك إجراء عمليات حسابية مباشرة على الأعداد:

| العملية | الرمز | مثال |
|---------|--------|-------|
| الجمع | + | `5 + 3` يساوي 8 |
| الطرح | - | `10 - 4` يساوي 6 |
| الضرب | * | `7 * 2` يساوي 14 |
| القسمة | / | `15 / 3` يساوي 5.0 (دائماً ناتج عشري) |
| القسمة الصحيحة | // | `15 // 4` يساوي 3 (التقسيم بدون باقي) |
| باقي القسمة | % | `15 % 4` يساوي 3 |
| الأس | ** | `2 ** 3` يساوي 8 |

مثال:

```python
a = 10
b = 3

print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.3333333333333335
print(a // b)  # 3
print(a % b)   # 1
print(a ** b)  # 1000
```

---

## 4. تحويل أنواع الأعداد

يمكنك تحويل الأعداد من نوع إلى آخر:

```python
a = 10     # int
b = float(a)  # تحويل إلى float
print(b)   # 10.0
print(type(b))  # <class 'float'>

c = int(3.7)  # تحويل float إلى int (يقطع الجزء العشري)
print(c)    # 3

d = complex(a)  # تحويل int إلى عدد مركب
print(d)    # (10+0j)
```

## 5. دوال مهمة للعمل مع الأعداد

- `abs(x)` : القيمة المطلقة للعدد x  
- `round(x, n)` : تقريب العدد x إلى n منازل عشرية  
- `pow(x, y)` : رفع x للأس y (مثال: pow(2,3) = 8)  
- `divmod(x, y)` : تُعيد قسمة صحيحة وباقي القسمة كزوج (مثال: divmod(15,4) = (3, 3))

مثال:

```python
print(abs(-7))          # 7
print(round(3.14159, 2)) # 3.14
print(pow(2, 5))        # 32
print(divmod(17, 3))    # (5, 2)
```
