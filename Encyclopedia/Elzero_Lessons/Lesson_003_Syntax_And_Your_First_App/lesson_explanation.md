## 1. ما هو التركيب النحوي (Syntax) في بايثون؟

التركيب النحوي هو مجموعة القواعد التي يجب اتباعها عند كتابة الكود حتى يفهمه المترجم (Interpreter) ويشغله بدون أخطاء.

مثلاً:
- يجب عليك كتابة الأوامر بشكل صحيح.
- استخدام علامات الترقيم في مكانها.
- تنظيم الكود (مثل المسافات البادئة).

إذا لم تتبع هذه القواعد، ستظهر لك أخطاء ولن يعمل البرنامج.

---

## 2. الكتابة الأساسية في بايثون

### طباعة النصوص على الشاشة

لإظهار نص أو رسالة على الشاشة، نستخدم الدالة `print()`.

مثال:

```python
print("مرحبًا بالعالم!")
```

**شرح:**  
- `print` هي دالة تُستخدم لطباعة النصوص أو القيم.  
- النصوص تُكتب بين علامتي اقتباس `" "` أو `' '`.

### المتغيرات وتخزين البيانات

المتغير هو مكان لتخزين البيانات، مثل الأرقام أو النصوص.

مثال:

```python
name = "أحمد"
age = 20
print("اسمي هو", name, "وعُمري", age, "سنة.")
```

**شرح:**  
- `name` هو اسم متغير يخزن نص.  
- `age` يخزن عدد صحيح.  
- نستخدم `print` لطباعة عدة أجزاء معًا، مفصولة بفواصل.

---

## 3. قواعد التركيب النحوي الأساسية

### - المسافات البادئة (Indentation)

في بايثون، المسافات البادئة مهمة جدًا لتحديد "كتل التعليمات" مثل داخل الحلقات أو الشروط.

مثال:

```python
if age >= 18:
    print("أنت بالغ.")
else:
    print("أنت قاصر.")
```

**ملاحظة:**  
بعد سطر الشرط `if`، يجب ترك مسافة بادئة (4 مسافات أو Tab) للكتلة التي تتبع الشرط.

### - التعليقات

يمكنك كتابة تعليقات لشرح الكود ولا يؤثر على تشغيل البرنامج. تبدأ التعليقات بـ `#`.

مثال:

```python
# هذا تعليق يشرح السطر التالي
print("مرحبا!")
```

---

## 4. كتابة أول برنامج كامل

لنكتب برنامجًا صغيرًا يسألك عن اسمك ويطبع رسالة ترحيب.

```python
# نطلب من المستخدم إدخال اسمه
name = input("ما اسمك؟ ")

# نطبع رسالة ترحيب باستخدام الاسم
print("مرحبًا يا " + name + "! سعيد بلقائك.")
```

**شرح:**  
- الدالة `input()` تأخذ مدخل من المستخدم كنص.  
- نستخدم علامة `+` لدمج النصوص معًا.

---

## 5. ملخص الدرس

- التركيب النحوي هو قواعد كتابة الكود صحيحة.  
- `print()` تطبع النصوص على الشاشة.  
- المتغيرات تخزن البيانات.  
- المسافات البادئة مهمة جدًا في بايثون.  
- التعليقات تبدأ بـ `#`.  
- استعملنا `input()` لأخذ بيانات من المستخدم.
