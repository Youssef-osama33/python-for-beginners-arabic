### ما هي السلاسل النصية (Strings)؟

في بايثون، **السلسلة النصية** هي نوع من البيانات يُستخدم لتخزين النصوص، مثل الكلمات، الجمل، أو أي مجموعة من الأحرف.

مثلاً:
```python
name = "أحمد"
greeting = 'مرحبا بك'
```

يمكن تعريف السلاسل النصية باستخدام علامات اقتباس مفردة `'...'` أو مزدوجة `"..."`، وكلاهما يعمل بنفس الطريقة.

---

### إنشاء السلاسل النصية

```python
# باستخدام علامات اقتباس مزدوجة
text1 = "مرحبا بالعالم"

# باستخدام علامات اقتباس مفردة
text2 = 'بايثون ممتع'

print(text1)
print(text2)
```

---

### الوصول إلى الأحرف داخل السلسلة النصية (Indexing)

كل حرف في السلسلة النصية له رقم ترتيب يُسمى **المؤشر (Index)**، يبدأ من 0.

مثلاً في النص `"مرحبا"`:

- 'م' موقعه 0
- 'ر' موقعه 1
- 'ح' موقعه 2
- وهكذا.

يمكنك الوصول إلى أي حرف باستخدام الأقواس المربعة `[]`:

```python
word = "مرحبا"
print(word[0])  # م
print(word[3])  # ب
```

يمكنك أيضًا استخدام الأرقام السالبة للعد من النهاية:

```python
print(word[-1])  # ا (آخر حرف)
print(word[-2])  # ب
```

---

### قص السلاسل النصية (Slicing)

يمكنك استخراج جزء من النص باستخدام **قص السلاسل**:

```python
text = "بايثون"
print(text[0:3])  # باي   (يأخذ الحروف من 0 حتى 2)
print(text[2:])   # ثون   (من الحرف 2 حتى النهاية)
print(text[:4])   # بايث   (من البداية حتى الحرف 3)
print(text[:])    # بايثون (النص كامل)
```

---

### دمج السلاسل النصية (Concatenation)

يمكنك دمج سلسلتين نصيتين باستخدام علامة الجمع `+`:

```python
first = "مرحبا"
second = " بالعالم"
full = first + second
print(full)  # مرحبا بالعالم
```

---

### تكرار السلاسل النصية (Repetition)

يمكنك تكرار النص عدة مرات باستخدام علامة الضرب `*`:

```python
text = "بايثون "
print(text * 3)  # بايثون بايثون بايثون 
```

---

### الطول (Length) - حساب عدد الأحرف

باستخدام الدالة `len()` يمكنك معرفة عدد الأحرف في السلسلة:

```python
word = "مرحبا"
print(len(word))  # 5
```

---

### بعض الدوال الشائعة للتعامل مع النصوص

- `lower()` لتحويل النص إلى حروف صغيرة:

```python
print("PYTHON".lower())  # python
```

- `upper()` لتحويل النص إلى حروف كبيرة:

```python
print("python".upper())  # PYTHON
```

- `strip()` لحذف الفراغات من البداية والنهاية:

```python
text = "   بايثون   "
print(text.strip())  # بايثون
```

- `replace()` لاستبدال جزء من النص:

```python
text = "مرحبا بالعالم"
new_text = text.replace("العالم", "بايثون")
print(new_text)  # مرحبا بايثون
```

- `split()` لتقسيم النص إلى قائمة كلمات بناءً على فاصل (مثلاً فراغ):

```python
sentence = "تعلم بايثون ممتع"
words = sentence.split()
print(words)  # ['تعلم', 'بايثون', 'ممتع']
```

---

### مثال كامل

```python
name = "علي"
greeting = "مرحبا " + name + "!"
print(greeting)  # مرحبا علي!

print("عدد حروف الاسم:", len(name))  # 3

print("الحرف الأول في الاسم:", name[0])  # ع

print("الاسم بحروف كبيرة:", name.upper())  # علي (بالعربية لا يتغير عادة)

text = "   بايثون لغة رائعة   "
print(text.strip())  # "بايثون لغة رائعة"
```
