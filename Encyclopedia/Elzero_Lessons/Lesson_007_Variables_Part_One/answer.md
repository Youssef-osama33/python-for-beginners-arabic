## حل التحدي البرمجي - درس بايثون 7: "المتغيرات الجزء الأول"

### المطلوب:
كتابة برنامج يطلب من المستخدم:
- اسمه الأول (نص)
- عمره (عدد صحيح)
- طوله بالسنتمتر (عدد عشري)

ثم يعرض رسالة ترحيبية تحتوي هذه المعلومات.

---

### الحل:

```python
# 1. طلب اسم المستخدم (نص)
name = input("من فضلك، أدخل اسمك الأول: ")

# 2. طلب عمر المستخدم (عدد صحيح)
# استخدام int لتحويل النص المدخل إلى عدد صحيح
age = int(input("من فضلك، أدخل عمرك (عدد صحيح): "))

# 3. طلب طول المستخدم (عدد عشري)
# استخدام float لتحويل النص المدخل إلى عدد عشري
height = float(input("من فضلك، أدخل طولك بالسنتمتر (مثلاً 170.5): "))

# 4. عرض رسالة ترحيبية منسقة
print(f"مرحباً يا {name}! عمرك {age} سنة وطولك {height} سم.")
```

---

### شرح الكود خطوة بخطوة:

1. **طلب اسم المستخدم**:
   ```python
   name = input("من فضلك، أدخل اسمك الأول: ")
   ```
   - الدالة `input()` تطلب من المستخدم إدخال نص.
   - النص الذي يدخله المستخدم يتم تخزينه في المتغير `name`.
   - نوع المتغير هنا هو **نص** (string).

2. **طلب عمر المستخدم**:
   ```python
   age = int(input("من فضلك، أدخل عمرك (عدد صحيح): "))
   ```
   - أولاً، `input()` تطلب إدخال نص من المستخدم.
   - ثم نستخدم الدالة `int()` لتحويل النص إلى عدد صحيح.
   - هذا ضروري لأن العمر عدد صحيح وليس نص.
   - إذا أدخل المستخدم قيمة غير قابلة للتحويل إلى عدد صحيح، سيحدث خطأ (في دروس متقدمة يمكننا التعامل مع هذا الخطأ).

3. **طلب طول المستخدم**:
   ```python
   height = float(input("من فضلك، أدخل طولك بالسنتمتر (مثلاً 170.5): "))
   ```
   - نفس فكرة العمر، لكن هذه المرة نحول النص إلى عدد عشري باستخدام `float()`.
   - هذا مناسب للطول لأنه يمكن أن يحتوي على أعداد عشرية (مثلاً 170.5).

4. **عرض رسالة الترحيب**:
   ```python
   print(f"مرحباً يا {name}! عمرك {age} سنة وطولك {height} سم.")
   ```
   - نستخدم **f-string** (سلسلة نصية مهيأة) لدمج المتغيرات داخل النص بطريقة سهلة وواضحة.
   - يتم استبدال `{name}`, `{age}`, `{height}` بالقيم المخزنة في المتغيرات.

---

### نقاط مهمة:

- اختيار نوع المتغير مهم:
  - الاسم: نص `string`.
  - العمر: عدد صحيح `int`.
  - الطول: عدد عشري `float`.
- استخدام دوال التحويل `int()` و `float()` لتحويل إدخال المستخدم النصي إلى النوع المناسب.
- استخدام المتغيرات لتخزين البيانات وإعادة استخدامها في البرنامج.

---

### مثال عند التشغيل:

```
من فضلك، أدخل اسمك الأول: علي
من فضلك، أدخل عمرك (عدد صحيح): 25
من فضلك، أدخل طولك بالسنتمتر (مثلاً 170.5): 175.2
مرحباً يا علي! عمرك 25 سنة وطولك 175.2 سم.
```
