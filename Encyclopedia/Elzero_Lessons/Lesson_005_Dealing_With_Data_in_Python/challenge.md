# تحدي الدرس 5: Dealing With Data in Python

## ❓ السؤال

سؤال تحدي برمجي:

لديك ملف نصي يحتوي على بيانات درجات طلاب في مادة الرياضيات، حيث كل سطر يحتوي على اسم الطالب متبوعًا بفاصلة ثم درجته، مثل:

```
أحمد,85
سارة,92
محمد,78
ليلى,92
يوسف,85
```

اكتب برنامج بايثون يقوم بالآتي:
1. قراءة البيانات من الملف وتحويلها إلى هيكل بيانات مناسب.
2. حساب متوسط الدرجات لجميع الطلاب.
3. تحديد أعلى درجة وأسماء الطلاب الذين حصلوا عليها.
4. ترتيب الطلاب تنازليًا حسب درجاتهم، وإذا تساوت الدرجات، يتم ترتيبهم أبجديًا حسب الاسم.
5. طباعة النتائج بطريقة منظمة، تشمل قائمة الطلاب بالترتيب، المتوسط، وأعلى درجة مع أسماء الطلاب الذين حصلوا عليها.

تأكد من التعامل مع أي عدد من الطلاب في الملف، وأن يكون البرنامج مرنًا
