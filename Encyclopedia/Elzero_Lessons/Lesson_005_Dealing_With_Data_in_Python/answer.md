## السؤال باختصار:
- قراءة بيانات من ملف نصي يحتوي على اسم الطالب ودرجته.
- حساب متوسط الدرجات.
- إيجاد أعلى درجة ومن حصل عليها.
- ترتيب الطلاب تنازليًا حسب الدرجة، وإذا تساوت الدرجات، حسب الاسم أبجديًا.
- طباعة النتائج بطريقة منظمة.

## الخطوة 1: التحضير

نفترض أن لدينا ملفًا اسمه `grades.txt` يحتوي على البيانات كما في المثال:

```
أحمد,85
سارة,92
محمد,78
ليلى,92
يوسف,85
```

---

## الحل الكامل مع الشرح

```python
# 1. قراءة البيانات من الملف وتحويلها إلى هيكل بيانات مناسب

# سنستخدم قائمة من القواميس لتخزين البيانات، كل عنصر يحتوي على 'name' و 'grade'

students = []

# فتح الملف للقراءة
with open('grades.txt', 'r', encoding='utf-8') as file:
    for line in file:
        # إزالة الفراغات وأحرف نهاية السطر
        line = line.strip()
        if not line:
            continue  # تخطي الأسطر الفارغة إن وجدت
        
        # تقسيم السطر عند الفاصلة
        parts = line.split(',')
        if len(parts) != 2:
            # إذا لم يكن التنسيق صحيح، نتجاهل السطر (يمكن تحسين التعامل لاحقًا)
            continue
        
        name = parts[0].strip()
        try:
            grade = int(parts[1].strip())  # تحويل الدرجة إلى عدد صحيح
        except ValueError:
            # إذا لم يكن بالإمكان تحويل الدرجة إلى عدد صحيح، نطبع تحذير ونتجاهل السطر
            print(f"تحذير: الدرجة غير صالحة للطالب {name}")
            continue
        
        # إضافة الطالب إلى القائمة
        students.append({'name': name, 'grade': grade})

# 2. حساب متوسط الدرجات لجميع الطلاب
if len(students) == 0:
    print("لا توجد بيانات طلاب لعرضها.")
    exit()

total_grades = sum(student['grade'] for student in students)
average_grade = total_grades / len(students)

# 3. تحديد أعلى درجة وأسماء الطلاب الذين حصلوا عليها

max_grade = max(student['grade'] for student in students)
top_students = [student['name'] for student in students if student['grade'] == max_grade]

# 4. ترتيب الطلاب تنازليًا حسب درجاتهم، وإذا تساوت الدرجات، حسب الاسم أبجديًا

# نستخدم الدالة sorted مع مفتاح ترتيب مركب:
# - الدرجة بالسالب (للحصول على ترتيب تنازلي)
# - الاسم (ترتيب أبجدي تصاعدي)
sorted_students = sorted(students, key=lambda s: (-s['grade'], s['name']))

# 5. طباعة النتائج بطريقة منظمة

print("قائمة الطلاب مرتبة حسب الدرجات:")
print("-----------------------------")
print(f"{'الاسم':<10} | {'الدرجة':>5}")
print("-----------------------------")
for student in sorted_students:
    print(f"{student['name']:<10} | {student['grade']:>5}")

print("-----------------------------")
print(f"متوسط الدرجات: {average_grade:.2f}")
print(f"أعلى درجة: {max_grade}")
print("الطلاب الحاصلون على أعلى درجة:")
for name in top_students:
    print(f"- {name}")
```

## شرح الكود خطوة بخطوة

1. **قراءة الملف:**

   - نفتح الملف `'grades.txt'` بترميز UTF-8 لأنه يحتوي على أسماء عربية.
   - نقرأ الملف سطرًا بسطر.
   - لكل سطر، نزيل الفراغات ونقسم السطر على الفاصلة `,` لنحصل على اسم الطالب ودرجته.
   - نحول الدرجة إلى عدد صحيح.
   - نضيف بيانات الطالب إلى قائمة `students` على شكل قاموس يحتوي على مفتاحين: `'name'` و `'grade'`.

2. **حساب المتوسط:**

   - نجمع درجات جميع الطلاب باستخدام دالة `sum` مع تعبير توليدي.
   - نقسم المجموع على عدد الطلاب.

3. **إيجاد أعلى درجة وأسماء الطلاب الذين حصلوا عليها:**

   - نستخدم `max` للعثور على أعلى درجة بين الطلاب.
   - نبحث عن كل الطلاب الذين درجتهم تساوي هذه الدرجة ونخزن أسمائهم في قائمة `top_students`.
