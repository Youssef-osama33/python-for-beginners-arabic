## 1. ما هي البيانات في بايثون؟

البيانات هي المعلومات التي نتعامل معها في البرامج، مثل الأرقام، النصوص، القوائم، وغيرها. بايثون توفر لنا عدة أنواع بيانات أساسية تمكننا من تخزين هذه المعلومات.

### الأنواع الأساسية للبيانات:

| نوع البيانات | الوصف                              | مثال في بايثون         |
|--------------|-----------------------------------|-----------------------|
| **int**      | الأعداد الصحيحة                    | 10، -5، 0             |
| **float**    | الأعداد العشرية (أعداد ذات فاصلة) | 3.14، -0.5            |
| **str**      | النصوص (سلاسل الحروف)             | "مرحبا"، 'Python'     |
| **bool**     | القيم المنطقية (صح أو خطأ)         | True، False           |
| **list**     | قائمة من العناصر (مرتبة)           | [1, 2, 3]، ['a', 'b']|
| **tuple**    | مجموعة ثابتة من العناصر (غير قابلة للتغيير) | (1, 2), ('x', 'y')    |
| **dict**     | قاموس يحتوي على أزواج (مفتاح: قيمة) | {'name': 'Ali', 'age': 25} |

## 2. كيفية تخزين البيانات في المتغيرات

المتغير هو اسم نستخدمه لتخزين البيانات للوصول إليها لاحقاً.

```python
# تخزين عدد صحيح
age = 25

# تخزين عدد عشري
price = 19.99

# تخزين نص
name = "أحمد"

# تخزين قيمة منطقية
is_student = True
```

---

## 3. التعامل مع القوائم (Lists)

القوائم تسمح لنا بتخزين مجموعة من العناصر يمكن تعديلها.

```python
# إنشاء قائمة أعداد
numbers = [1, 2, 3, 4, 5]

# إضافة عنصر جديد
numbers.append(6)  # الآن القائمة: [1, 2, 3, 4, 5, 6]

# الوصول إلى عنصر عبر الفهرس (الفهرس يبدأ من 0)
print(numbers[0])  # يطبع 1

# تعديل عنصر
numbers[1] = 10
print(numbers)  # [1, 10, 3, 4, 5, 6]
```

---

## 4. التعامل مع القواميس (Dictionaries)

القواميس تخزن البيانات على شكل أزواج (مفتاح: قيمة).

```python
# إنشاء قاموس
person = {
    'name': 'سارة',
    'age': 30,
    'city': 'الرياض'
}

# الوصول إلى القيمة عبر المفتاح
print(person['name'])  # يطبع سارة

# تعديل قيمة
person['age'] = 31

# إضافة زوج جديد
person['job'] = 'مهندسة'

print(person)
```

## 5. قراءة البيانات من المستخدم (Input)

يمكننا طلب بيانات من المستخدم باستخدام الدالة `input()`.

```python
name = input("ما اسمك؟ ")
age = input("كم عمرك؟ ")

print("مرحبا " + name + "! عمرك " + age + " سنة.")
```

**ملاحظة:** البيانات التي تُدخل بواسطة `input()` تكون من النوع نص (str)، لذلك لتحويلها إلى رقم نستخدم:

```python
age = int(input("كم عمرك؟ "))  # تحويل إلى عدد صحيح
```

## 6. تحويل أنواع البيانات (Type Conversion)

في بعض الأحيان نحتاج لتحويل نوع البيانات إلى نوع آخر.

```python
num_str = "100"
num_int = int(num_str)  # تحويل نص إلى عدد صحيح

pi_str = "3.14"
pi_float = float(pi_str)  # تحويل نص إلى عدد عشري

age = 25
age_str = str(age)  # تحويل عدد إلى نص
```
