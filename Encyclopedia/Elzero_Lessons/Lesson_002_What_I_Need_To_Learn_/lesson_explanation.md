في هذا الدرس سنتعرف على أهم المفاهيم الأساسية التي تحتاجها لتصبح مبرمج بايثون محترفًا. تعلم البرمجة يشبه بناء منزل: تحتاج أولاً إلى الأساسات القوية قبل أن تنتقل إلى التفاصيل المعقدة. في بايثون، هذه الأساسات تشمل عدة مواضيع رئيسية سنتناولها مع أمثلة بسيطة وواضحة.

---

## 1. المتغيرات Variables

المتغير هو مكان لتخزين البيانات مثل الأرقام أو النصوص. يمكنك استخدام المتغيرات لتخزين المعلومات ثم استخدامها لاحقًا في برنامجك.

### مثال:

```python
name = "أحمد"
age = 20
print("مرحبًا، اسمي", name, "وعُمري", age, "سنة.")
```

في هذا المثال:
- `name` هو متغير يخزن نصًا.
- `age` هو متغير يخزن عددًا صحيحًا.
- `print` تطبع النص مع المتغيرات.

---

## 2. أنواع البيانات Data Types

بايثون يدعم عدة أنواع من البيانات الأساسية:

- **النصوص (Strings):** مثل `"مرحبا"`
- **الأعداد الصحيحة (Integers):** مثل `10`
- **الأعداد العشرية (Floats):** مثل `3.14`
- **القيم المنطقية (Booleans):** `True` أو `False`

### مثال:

```python
text = "مرحبا"
number = 10
pi = 3.14
is_sunny = True

print(text, number, pi, is_sunny)
```

---

## 3. العمليات الحسابية Arithmetic Operations

يمكنك استخدام بايثون لإجراء عمليات حسابية مثل الجمع، الطرح، الضرب والقسمة.

### مثال:

```python
a = 5
b = 3

print("الجمع:", a + b)       # 8
print("الطرح:", a - b)       # 2
print("الضرب:", a * b)       # 15
print("القسمة:", a / b)      # 1.666...
print("القسمة الصحيحة:", a // b)  # 1
print("الباقي:", a % b)      # 2
print("الأس:", a ** b)       # 125
```

---

## 4. الجمل الشرطية If Statements

تساعدك الجمل الشرطية في اتخاذ قرارات داخل البرنامج بناءً على تحقق شرط معين.

### مثال:

```python
age = 18

if age >= 18:
    print("أنت بالغ.")
else:
    print("أنت قاصر.")
```

---

## 5. الحلقات Loops

تستخدم الحلقات لتكرار تنفيذ جزء من الكود أكثر من مرة.

### مثال باستخدام حلقة for:

```python
for i in range(5):
    print("العدد هو:", i)
```

### مثال باستخدام حلقة while:

```python
count = 0
while count < 5:
    print("العدد هو:", count)
    count += 1
```

---

## 6. الدوال Functions

الدوال هي مجموعة من التعليمات التي يمكن إعادة استخدامها في البرنامج.

### مثال:

```python
def greet(name):
    print("مرحبًا يا", name)

greet("سارة")
greet("محمد")
```
