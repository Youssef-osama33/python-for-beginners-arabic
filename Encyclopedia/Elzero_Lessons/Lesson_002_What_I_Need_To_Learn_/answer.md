## المطلوب:

- نأخذ عدد المواد من المستخدم.
- نأخذ أسماء المواد واحدة تلو الأخرى.
- نخزنها في قائمة.
- نحذف التكرار (تترك المادة مرة واحدة فقط).
- نرتب القائمة أبجدياً.
- نعرض القائمة النهائية للمستخدم.

---

## الحل الكامل مع الشرح:

```python
# 1. نطلب من المستخدم إدخال عدد المواد التي يريد إضافتها
n = int(input("ادخل عدد المواد: "))

# 2. نهيئ قائمة فارغة لتخزين أسماء المواد
subjects = []

# 3. نستخدم حلقة for لكي نطلب من المستخدم إدخال أسماء المواد واحدة تلو الأخرى
for i in range(1, n + 1):
    subject = input(f"ادخل المادة {i}: ")
    subjects.append(subject)  # نضيف المادة إلى القائمة

# 4. لحذف المواد المكررة، يمكننا تحويل القائمة إلى مجموعة (set) حيث المجموعات لا تسمح بالتكرار
unique_subjects = set(subjects)

# 5. نريد ترتيب المواد أبجدياً، لذلك نحول المجموعة مرة أخرى إلى قائمة ونرتبها باستخدام الدالة sorted()
sorted_subjects = sorted(unique_subjects)

# 6. نعرض القائمة النهائية للمستخدم
print("\nالمواد التي تحتاج لتعلمها هي:")
for subject in sorted_subjects:
    print(subject)
```

---

## شرح الكود خطوة بخطوة:

1. `n = int(input("ادخل عدد المواد: "))`
   - نطلب من المستخدم إدخال عدد المواد.
   - `input()` تأخذ النص المدخل، و`int()` تحول النص إلى عدد صحيح.

2. `subjects = []`
   - ننشئ قائمة فارغة باسم `subjects` لتخزين أسماء المواد.

3. الحلقة `for i in range(1, n + 1):`
   - نكرر العملية `n` مرات، كل مرة نطلب اسم مادة.
   - `range(1, n + 1)` يبدأ من 1 حتى `n` (شاملة).

4. داخل الحلقة:
   - `subject = input(f"ادخل المادة {i}: ")` يأخذ اسم المادة من المستخدم.
   - `subjects.append(subject)` يضيف المادة إلى القائمة.

5. `unique_subjects = set(subjects)`
   - نحول القائمة إلى مجموعة `set` لإزالة التكرار.
   - المجموعات لا تسمح بالعناصر المكررة.

6. `sorted_subjects = sorted(unique_subjects)`
   - نحول المجموعة إلى قائمة مرتبة أبجدياً باستخدام `sorted()`.

7. نطبع رسالة للمستخدم:
   - `"المواد التي تحتاج لتعلمها هي:"`
   - ثم نطبع كل مادة في القائمة النهائية.
