## السؤال:
اكتب برنامج بايثون يقوم بالآتي:

1. يطلب من المستخدم إدخال جملة نصية تحتوي على كلمات متعددة.
2. يحول الجملة إلى صيغة تبدأ كل كلمة فيها بحرف كبير (Capitalized).
3. يستبدل كل ظهور لكلمة "Python" في الجملة بكلمة "بايثون" (مع مراعاة اختلاف حالة الأحرف، أي أن الكلمات "python" أو "PYTHON" أو أي شكل آخر يجب استبدالها أيضاً).
4. يحذف أي مسافات زائدة (أكثر من مسافة واحدة) بين الكلمات بحيث تبقى مسافة واحدة فقط.
5. في النهاية، يعرض الجملة المعدلة مع عدد الكلمات الموجودة فيها.

*ملاحظة:* استخدم فقط دوال السلاسل النصية التي تم تناولها في الدرس رقم 13.

---

## الحل الكامل مع الشرح:

```python
# 1. نطلب من المستخدم إدخال جملة نصية
sentence = input("أدخل جملة نصية تحتوي على كلمات متعددة: ")

# 2. نحول الجملة إلى صيغة تبدأ كل كلمة فيها بحرف كبير
# نستخدم الدالة title() التي تحول أول حرف من كل كلمة إلى كبير
sentence = sentence.title()

# 3. نستبدل كل ظهور لكلمة "Python" (بأي حالة) بكلمة "بايثون"
# لكن الدالة replace() لا تفرق بين الحالات (case-sensitive)، لذلك نحتاج لطريقة أخرى.

# الحل: 
# - نحول الجملة كلها إلى حروف صغيرة مؤقتاً لكي نجد كلمة "python" بسهولة
# - ثم نستبدل "python" بـ "بايثون"
# - لكن لأننا نريد الحفاظ على حالة الحروف الأصلية (حيث أن sentence الآن مكتوبة بحروف كبيرة بداية كل كلمة)
# - فنقوم باستبدال على الجملة المعدلة (title case) باستخدام replace مع الحالة الصحيحة.

# الطريقة الأبسط هنا: بما أن الجملة أصبحت بحروف كبيرة أولية (title case)، 
# كلمة "Python" ستظهر بهذه الصيغة فقط (P كابيتال، ython صغيرة).
# لذلك نستبدل "Python" فقط.

sentence = sentence.replace("Python", "بايثون")

# ملاحظة: إذا كان في الجملة كلمة "PYTHON" أو "python" أو غيرها كما في النص الأصلي،
# فبعد title() ستصبح "Python"، لذلك يكفي استبدال "Python".

# 4. نحذف أي مسافات زائدة بين الكلمات بحيث تبقى مسافة واحدة فقط
# الطريقة: 
# - نقسم الجملة إلى كلمات (split() بدون معاملات يفصل على أي عدد من المسافات)
# - ثم نعيد الربط بينهم بمسافة واحدة فقط (join)

words = sentence.split()  # تفصل الكلمات وتزيل المسافات الزائدة
sentence = " ".join(words)  # تعيد دمج الكلمات بمسافة واحدة

# 5. نعرض الجملة المعدلة مع عدد الكلمات الموجودة فيها
# عدد الكلمات = طول قائمة الكلمات التي حصلنا عليها بعد التقسيم
word_count = len(words)

print("الجملة المعدلة:")
print(sentence)
print(f"عدد الكلمات في الجملة: {word_count}")
```

## شرح الكود خطوة بخطوة:

1. **إدخال الجملة:**
   ```python
   sentence = input("أدخل جملة نصية تحتوي على كلمات متعددة: ")
   ```
   نستخدم الدالة `input()` لطلب جملة نصية من المستخدم.

2. **تحويل كل كلمة لتبدأ بحرف كبير:**
   ```python
   sentence = sentence.title()
   ```
   الدالة `title()` تقوم بتحويل أول حرف من كل كلمة إلى حرف كبير، والباقي إلى صغير.

3. **استبدال كلمة "Python" بكلمة "بايثون" بغض النظر عن حالة الأحرف:**
   - بعد تطبيق `title()`، أي كلمة "python" بأي حالة ستصبح "Python".
   - لذلك نستخدم:
     ```python
     sentence = sentence.replace("Python", "بايثون")
     ```
   - الدالة `replace()` تستبدل كل ظهور لكلمة "Python" بكلمة "بايثون".
