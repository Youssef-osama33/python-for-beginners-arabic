## 1. طريقة `split()`

- **الغرض**: تقسيم النص إلى قائمة من الكلمات أو الأجزاء بناءً على فاصل معين.
- **الصيغة**: `string.split(separator, maxsplit)`
  - `separator` (اختياري): الحرف أو السلسلة التي نريد استخدامها كفاصل. إذا لم نحددها، يكون الفاصل هو الفراغ.
  - `maxsplit` (اختياري): عدد التقسيمات التي نريد إجراؤها.

### مثال:

```python
text = "مرحبا بك في عالم بايثون"
words = text.split()
print(words)
```

**الناتج:**

```
['مرحبا', 'بك', 'في', 'عالم', 'بايثون']
```

### مثال مع فاصل مخصص:

```python
data = "apple,banana,cherry"
fruits = data.split(',')
print(fruits)
```

**الناتج:**

```
['apple', 'banana', 'cherry']
```

## 2. طريقة `join()`

- **الغرض**: دمج قائمة من الكلمات أو العناصر إلى سلسلة نصية واحدة، مع وضع فاصل معين بينها.
- **الصيغة**: `separator.join(iterable)`
  - `separator`: النص الذي نريد وضعه بين العناصر (مثل فراغ، فاصلة، إلخ).
  - `iterable`: قائمة أو أي مجموعة من النصوص.

### مثال:

```python
words = ['مرحبا', 'بك', 'في', 'بايثون']
sentence = ' '.join(words)
print(sentence)
```

**الناتج:**

```
مرحبا بك في بايثون
```

---

## 3. طريقة `strip()`

- **الغرض**: إزالة الفراغات أو أحرف معينة من بداية ونهاية النص.
- **الصيغة**: `string.strip(chars)`
  - `chars` (اختياري): مجموعة الأحرف التي نريد إزالتها. إذا لم نحددها، يتم إزالة الفراغات فقط.

### مثال:

```python
text = "   مرحبا بك   "
clean_text = text.strip()
print(clean_text)
```

**الناتج:**

```
مرحبا بك
```

### مثال على إزالة أحرف معينة:

```python
text = "###مرحبا###"
clean_text = text.strip('#')
print(clean_text)
```

**الناتج:**

```
مرحبا
```

## 4. طريقة `replace()`

- **الغرض**: استبدال جزء معين من النص بجزء آخر.
- **الصيغة**: `string.replace(old, new, count)`
  - `old`: النص القديم الذي نريد استبداله.
  - `new`: النص الجديد.
  - `count` (اختياري): عدد الاستبدالات التي نريد إجراؤها.

### مثال:

```python
text = "أنا أحب البرمجة، البرمجة ممتعة"
new_text = text.replace("البرمجة", "بايثون")
print(new_text)
```

**الناتج:**

```
أنا أحب بايثون، بايثون ممتعة
```

## 5. طريقة `startswith()` و `endswith()`

- **الغرض**: التحقق إذا كانت السلسلة تبدأ أو تنتهي بنص معين.
- **الصيغة**:
  - `string.startswith(prefix)`
  - `string.endswith(suffix)`

### مثال:

```python
text = "مرحبا بك في بايثون"
print(text.startswith("مرحبا"))  # True
print(text.endswith("بايثون"))   # True
print(text.startswith("أهلا"))   # False
```

---

## 6. طريقة `lower()` و `upper()`

- **الغرض**: تحويل النص كله إلى حروف صغيرة أو كبيرة.
- **الصيغة**:
  - `string.lower()` لتحويل النص إلى أحرف صغيرة.
  - `string.upper()` لتحويل النص إلى أحرف كبيرة.

### مثال:

```python
text = "Python Is Fun"
print(text.lower())  # python is fun
print(text.upper())  # PYTHON IS FUN
```

- **الصيغة**: `string.count(substring)`
