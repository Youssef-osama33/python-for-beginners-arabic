## حل السؤال:

```python
# استيراد مكتبة string للتعامل مع علامات الترقيم
import string

# 1. نطلب من المستخدم إدخال جملة نصية
sentence = input("أدخل جملة نصية: ")

# 2. نحول الجملة إلى حروف صغيرة
sentence_lower = sentence.lower()

# 3. نزيل علامات الترقيم
# نقوم بإنشاء جدول ترجمة يحول كل علامة ترقيم إلى None (أي حذفها)
translator = str.maketrans('', '', string.punctuation + "؟،؛.!")

sentence_no_punct = sentence_lower.translate(translator)

# 4. نقسم الجملة إلى كلمات باستخدام split (تفصل على الفراغات)
words = sentence_no_punct.split()

# 5. نحسب عدد الكلمات المختلفة وعدد تكرار كل كلمة
word_counts = {}

for word in words:
    if word in word_counts:
        word_counts[word] += 1
    else:
        word_counts[word] = 1

# 6. نرتب الكلمات أبجدياً
sorted_words = sorted(word_counts.items(), key=lambda x: x[0])

# 7. نطبع الكلمات مع عدد التكرار لكل كلمة
for word, count in sorted_words:
    print(f"{word}: {count}")
```

## شرح الكود خطوة بخطوة:

1. **استيراد مكتبة string**  
   مكتبة `string` تحتوي على مجموعة من ثوابت مفيدة، منها `string.punctuation` التي تحتوي على علامات الترقيم الإنجليزية.

2. **إدخال الجملة**  
   نستخدم `input()` لكي يكتب المستخدم جملة نصية.

3. **تحويل الحروف إلى صغيرة**  
   نستخدم دالة `lower()` لتحويل كل الأحرف إلى حروف صغيرة لتوحيد شكل الكلمات (مثلاً "مرحباً" و "مرحباً" بالحروف الكبيرة ستكون نفس الكلمة).

4. **إزالة علامات الترقيم**  
   - نستخدم `str.maketrans()` لإنشاء جدول ترجمة يحذف علامات الترقيم.  
   - نضيف علامات الترقيم العربية مثل: "؟،؛.!" لأنها غير موجودة في `string.punctuation`.  
   - `translate()` تطبق هذا الجدول على الجملة وتحذف هذه العلامات.

5. **تقسيم الجملة إلى كلمات**  
   باستخدام `split()` نفصل النص إلى كلمات بناءً على الفراغات.

6. **حساب تكرار الكلمات**  
   - نستخدم قاموس `word_counts` حيث المفتاح هو الكلمة والقيمة هي عدد تكرارها.  
   - إذا كانت الكلمة موجودة نزيد العد، وإلا نضيفها مع القيمة 1.

7. **ترتيب الكلمات أبجدياً**  
   - نستخدم `sorted()` لترتيب عناصر القاموس (مفاتيح وقيم) حسب الكلمة (المفتاح).

8. **طباعة النتائج**  
   - نطبع كل كلمة مع عدد تكرارها بطريقة واضحة.
