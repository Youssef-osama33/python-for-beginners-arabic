# درس بايثون رقم 10: **الدمج (Concatenation) والتكرار (Training)**

في هذا الدرس سنتعلم:

1. **الدمج (Concatenation)**: كيفية دمج النصوص (السلاسل النصية) أو القوائم أو غيرها من البيانات معًا.
2. **التكرار (Training)**: كيفية تكرار النصوص أو القوائم باستخدام عامل التكرار.

---

## 1. الدمج (Concatenation)

### ما هو الدمج؟

الدمج يعني ربط سلسلتين نصيتين أو أكثر معًا ليصبحوا نصًا واحدًا طويلًا.

### في بايثون:

- نستخدم العلامة `+` لدمج النصوص.
- يمكن أيضًا دمج القوائم باستخدام `+`.

### أمثلة:

#### دمج نصوص (Strings)

```python
# تعريف سلسلتين نصيتين
text1 = "مرحبا"
text2 = " بالعالم"

# دمج النصوص
result = text1 + text2

print(result)  # النتيجة: مرحبا بالعالم
```

#### دمج قوائم (Lists)

```python
list1 = [1, 2, 3]
list2 = [4, 5, 6]

# دمج القوائم
merged_list = list1 + list2

print(merged_list)  # النتيجة: [1, 2, 3, 4, 5, 6]
```

---

## 2. التكرار (Training)

### ما هو التكرار؟

التكرار يعني تكرار النص أو القائمة عدة مرات.

### في بايثون:

- نستخدم العلامة `*` لتكرار النصوص أو القوائم.
- نحدد عدد مرات التكرار على يمين العلامة.

### أمثلة:

#### تكرار نص (String)

```python
word = "بايثون "

# تكرار النص 3 مرات
repeated = word * 3

print(repeated)  # النتيجة: بايثون بايثون بايثون 
```

#### تكرار قائمة (List)

```python
numbers = [1, 2]

# تكرار القائمة 4 مرات
repeated_list = numbers * 4

print(repeated_list)  # النتيجة: [1, 2, 1, 2, 1, 2, 1, 2]
```

---

## ملخص

| العملية          | الرمز | مثال                         | النتيجة                 |
|------------------|-------|------------------------------|-------------------------|
| دمج نصوص         | +     | "مرحبا" + " بالعالم"          | "مرحبا بالعالم"         |
| دمج قوائم        | +     | [1, 2] + [3, 4]               | [1, 2, 3, 4]            |
| تكرار نص         | *     | "بايثون " * 3                 | "بايثون بايثون بايثون " |
| تكرار قائمة      | *     | [1, 2] * 3                   | [1, 2, 1, 2, 1, 2]      |

---

## أسئلة تحدي

1. اكتب برنامجًا يجمع بين اسمك الأول واسم عائلتك باستخدام الدمج ثم يكرر الناتج مرتين.
2. لديك قائمتان: `a = [10, 20]` و `b = [30, 40]`. ادمجهما ثم كرر الناتج 3 مرات، ما النتيجة؟
3. لماذا لا يمكن دمج عددين صحيحين باستخدام `+` بنفس الطريقة التي ندمج بها النصوص؟
