# تحدي الدرس 6: Some Data Types Overview

## ❓ السؤال

سؤال تحدي برمجي:

اكتب برنامجًا بلغة بايثون يقوم بإنشاء قائمة تحتوي على عناصر من أنواع بيانات مختلفة (مثل: عدد صحيح، عدد عشري، نص، قائمة فرعية، وقيمة منطقية). ثم، يقوم البرنامج بما يلي:

1. يطبع نوع كل عنصر في القائمة.
2. يعدّ عدد العناصر التي هي من نوع رقمي (int أو float).
3. يحول كل عنصر نصي إلى أحرف كبيرة (uppercase).
4. يضيف عنصر جديد من نوع بيانات مختلف إلى القائمة.
5. يطبع القائمة بعد التعديلات.

استخدم المفاهيم الأساسية لأنواع البيانات في بايثون لتعزيز فهمك.
