## 📚 شرح الدرس

درس بايثون رقم 6: نظرة عامة على بعض أنواع البيانات (Some Data Types Overview)

مقدمة:
في هذا الدرس سنتعرف على بعض أنواع البيانات الأساسية في لغة بايثون. فهم أنواع البيانات يساعدنا على التعامل مع المعلومات بشكل صحيح داخل البرنامج. سنتناول أهم الأنواع مثل الأعداد الصحيحة، الأعداد العشرية، النصوص، القوائم، والقيم المنطقية.

1. الأعداد الصحيحة (Integers)
هي أعداد بدون فاصلة عشرية، مثل: 3، -7، 1000.
مثال:
```python
a = 10
b = -5
print(a)
print(b)
```
الناتج:
```
10
-5
```

2. الأعداد العشرية (Floating Point Numbers)
هي أعداد تحتوي على فاصلة عشرية، مثل: 3.14، -0.5، 2.0.
مثال:
```python
x = 3.14
y = -0.5
print(x)
print(y)
```
الناتج:
```
3.14
-0.5
```

3. السلاسل النصية (Strings)
هي نصوص محاطة بعلامات اقتباس مفردة أو مزدوجة.
مثال:
```python
name = "محمد"
greeting = 'مرحبا بك'
print(name)
print(greeting)
```
الناتج:
```
محمد
مرحبا بك
```

4. القوائم (Lists)
هي مجموعات مرتبة من العناصر يمكن أن تحتوي على أنواع بيانات مختلفة.
مثال:
```python
fruits = ["تفاح", "موز", "برتقال"]
print(fruits)
print(fruits[0])  # العنصر الأول
```
الناتج:
```
['تفاح', 'موز', 'برتقال']
تفاح
```

5. القيم المنطقية (Boolean)
تمثل إما True (صحيح) أو False (خطأ). تستخدم في شروط وبرمجة التحكم.
مثال:
```python
is_sunny = True
is_raining = False
print(is_sunny)
print(is_raining)
```
الناتج:
```
True
False
```

ملخص:
- int للأعداد الصحيحة.
- float للأعداد العشرية.
- str للسلاسل النصية.
- list للقوائم.
- bool للقيم المنطقية.

أسئلة تحدي:
1. عرف متغير باسم age يحتوي على عدد صحيح يمثل عمرك.
2. اصنع قائمة بها ثلاثة ألوان مفضلة لديك.
3. اكتب نص ترحيبي باسمك في متغير من نوع نصي.
4. اكتب متغير boolean يدل على إذا كنت طالبًا أم لا.

حلول الأسئلة:
```python
age = 25
colors = ["أحمر", "أزرق", "أخضر"]
welcome_message = "مرحباً، اسمي علي"
is_student = True
```
