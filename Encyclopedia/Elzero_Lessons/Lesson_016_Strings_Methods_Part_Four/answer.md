## السؤال:
اكتب برنامجًا بلغة بايثون يطلب من المستخدم إدخال جملة نصية تحتوي على كلمات وأرقام ورموز خاصة. ثم يقوم البرنامج بما يلي:

1. يستخرج جميع الكلمات التي تبدأ بحرف كبير (Capitalized words) ويجمعها في قائمة منفصلة.
2. يحسب عدد الأرقام الموجودة في الجملة (كل رقم مفرد منفصل).
3. يستبدل جميع الرموز الخاصة (أي حرف ليس بحرف أو رقم أو فراغ) في الجملة بعلامة النجمة '*'.
4. يعرض للمستخدم:
   - قائمة الكلمات التي تبدأ بحرف كبير.
   - عدد الأرقام.
   - الجملة المعدلة بعد استبدال الرموز الخاصة.

## الحل البرمجي مع الشرح خطوة بخطوة:

```python
# نطلب من المستخدم إدخال جملة نصية
sentence = input("أدخل جملة نصية تحتوي على كلمات وأرقام ورموز خاصة:\n")

# 1. استخراج الكلمات التي تبدأ بحرف كبير
# نفصل الجملة إلى كلمات باستخدام split()
words = sentence.split()

# قائمة لتخزين الكلمات التي تبدأ بحرف كبير
capitalized_words = []

for word in words:
    # نتحقق إذا كانت الكلمة غير فارغة وبدأت بحرف كبير
    # isalpha() للتأكد أن الحرف هو حرف وليس رقم أو رمز
    # .istitle() تعطي True للكلمة التي تبدأ بحرف كبير والباقي صغير (لكن هنا نحتاج فقط أول حرف كبير)
    # لذا نستخدم isupper() على أول حرف
    if len(word) > 0 and word[0].isupper():
        capitalized_words.append(word)

# 2. حساب عدد الأرقام المفردة في الجملة
# نمر على كل حرف في الجملة ونتحقق هل هو رقم باستخدام isdigit()
digit_count = 0
for char in sentence:
    if char.isdigit():
        digit_count += 1

# 3. استبدال الرموز الخاصة بعلامة النجمة '*'
# الرموز الخاصة هي أي حرف ليس بحرف (a-z,A-Z)، رقم (0-9) أو فراغ
# نمر على كل حرف ونبني جملة جديدة
modified_sentence = ""

for char in sentence:
    # نتحقق إذا كان الحرف حرفًا (isalpha) أو رقم (isdigit) أو فراغ (char == ' ')
    if char.isalpha() or char.isdigit() or char == ' ':
        # نضيف الحرف كما هو
        modified_sentence += char
    else:
        # إذا كان رمز خاص، نضيف '*'
        modified_sentence += '*'

# 4. عرض النتائج
print("\nالكلمات التي تبدأ بحرف كبير هي:")
print(capitalized_words)

print("\nعدد الأرقام في الجملة هو:", digit_count)

print("\nالجملة بعد استبدال الرموز الخاصة بعلامة '*':")
print(modified_sentence)
```

## شرح الكود خطوة بخطوة:

1. **إدخال الجملة:**

   - نستخدم الدالة `input()` لطلب جملة نصية من المستخدم.

2. **استخراج الكلمات التي تبدأ بحرف كبير:**

   - نستخدم طريقة `.split()` لتقسيم الجملة إلى كلمات بناءً على الفراغات.
   - نمر على كل كلمة، ونتحقق من أول حرف فيها باستخدام `word[0].isupper()`:
     - `isupper()` تعني أن الحرف كبير.
   - إذا كان أول حرف كبير، نضيف الكلمة إلى القائمة `capitalized_words`.
   
3. **حساب عدد الأرقام:**

   - نمر على كل حرف في الجملة باستخدام حلقة `for`.
   - نستخدم `char.isdigit()` للتحقق ما إذا كان الحرف رقمًا.
   - إذا كان رقمًا، نزيد العداد `digit_count` بواحد.

4. **استبدال الرموز الخاصة بعلامة '*' :**

   - نمر على كل حرف في الجملة.
   - نتحقق هل الحرف:
     - حرف (باستخدام `char.isalpha()`)
     - رقم (`char.isdigit()`)
     - أو فراغ (`char == ' '`)
   - إذا كان كذلك، نضيف الحرف كما هو إلى النص الجديد `modified_sentence`.
   - إذا لم يكن كذلك (أي رمز خاص)، نضيف `'*'`.

5. **طباعة النتائج:**

   - نطبع قائمة الكلمات التي تبدأ بحرف كبير.
   - نطبع عدد الأرقام.
   - نطبع الجملة بعد تعديلها.
