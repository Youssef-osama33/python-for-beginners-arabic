## مقدمة

السلاسل النصية (Strings) في بايثون هي عبارة عن مجموعة من الأحرف، مثل الكلمات أو الجمل. بايثون توفر طرق (Methods) كثيرة للتعامل مع النصوص، مثل تعديل النص، البحث، التحقق من نوع الأحرف، وغيرها.

في هذا الجزء الرابع، سنتناول الطرق التالية:

1. `.startswith()`
2. `.endswith()`
3. `.join()`
4. `.partition()`
5. `.rpartition()`
6. `.zfill()`
7. `.translate()`

---

## 1. طريقة startswith()

تستخدم للتحقق إذا كان النص يبدأ بسلسلة معينة.

**الصيغة:**
```python
string.startswith(substring)
```

- ترجع `True` إذا بدأ النص بـ `substring`، و`False` إذا لم يبدأ به.

**مثال:**
```python
text = "مرحبا بالعالم"
print(text.startswith("مرحبا"))  # True
print(text.startswith("بالعالم"))  # False
```

---

## 2. طريقة endswith()

تستخدم للتحقق إذا كان النص ينتهي بسلسلة معينة.

**الصيغة:**
```python
string.endswith(substring)
```

- ترجع `True` إذا انتهى النص بـ `substring`، و`False` إذا لم ينته به.

**مثال:**
```python
text = "مرحبا بالعالم"
print(text.endswith("العالم"))  # True
print(text.endswith("مرحبا"))  # False
```

---

## 3. طريقة join()

تستخدم لدمج مجموعة من النصوص (مثل قائمة) إلى نص واحد، مع وضع فاصل بين النصوص.

**الصيغة:**
```python
separator.join(iterable)
```

- `separator` هو النص الذي سيتم وضعه بين العناصر.
- `iterable` هي مجموعة نصوص مثل قائمة.

**مثال:**
```python
words = ["أنا", "أحب", "بايثون"]
result = " ".join(words)
print(result)  # أنا أحب بايثون

result2 = "-".join(words)
print(result2)  # أنا-أحب-بايثون
```

---

## 4. طريقة partition()

تقسم النص إلى 3 أجزاء بناءً على وجود سلسلة معينة:

- الجزء قبل السلسلة.
- السلسلة نفسها.
- الجزء بعد السلسلة.

**الصيغة:**
```python
string.partition(separator)
```

- إذا لم توجد السلسلة، يرجع النص كاملاً في الجزء الأول، والأجزاء الثانية والثالثة تكون فارغة.

**مثال:**
```python
text = "مرحبا بالعالم"
parts = text.partition("ب")
print(parts)  # ('مرح', 'ب', 'ا بالعالم')
```

---

## 5. طريقة rpartition()

تشبه `partition()` لكنها تبدأ البحث من نهاية النص.

**الصيغة:**
```python
string.rpartition(separator)
```

**مثال:**
```python
text = "أحب التفاح والتفاح أحلى"
parts = text.rpartition("تفاح")
print(parts)  # ('أحب التفاح و', 'تفاح', ' أحلى')
```

---

## 6. طريقة zfill()

تستخدم لملء النص بالأصفار من جهة اليسار ليصل لطول معين.

**الصيغة:**
```python
string.zfill(width)
```

- `width` هو الطول النهائي للنص.

**مثال:**
```python
number = "42"
print(number.zfill(5))  # 00042
```

---

## 7. طريقة translate()

تستخدم لتحويل الأحرف في النص باستخدام جدول تحويل (ترجمة).

عادةً ما تستخدم مع دالة `str.maketrans()` التي تحدد التحويلات.

**الصيغة:**
```python
string.translate(table)
```

**مثال:**
```python
table = str.maketrans("abc", "123")
text = "abcabc"
print(text.translate(table))  # 123123
```
