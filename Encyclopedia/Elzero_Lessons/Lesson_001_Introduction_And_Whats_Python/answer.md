## كود البرنامج الكامل:

```python
# 1. نطلب من المستخدم إدخال اسمه بالكامل
name = input("من فضلك، أدخل اسمك الكامل (الاسم + اسم العائلة): ")

# 2. نحتاج إلى الحصول على عمر المستخدم كعدد صحيح موجب
while True:
    age_input = input("من فضلك، أدخل عمرك (رقم صحيح موجب): ")
    # 3. نتحقق من أن العمر هو رقم صحيح موجب
    if age_input.isdigit():
        age = int(age_input)
        if age > 0:
            break  # إذا كان العمر صحيحًا، نخرج من الحلقة
        else:
            print("الرجاء إدخال رقم موجب.")
    else:
        print("الرجاء إدخال رقم صحيح فقط.")

# 4. نعرض رسالة ترحيب مع الاسم وعدد السنوات المتبقية حتى 100 سنة
if age < 100:
    years_left = 100 - age
    print(f"مرحبًا {name}! بقي لك {years_left} سنة حتى تبلغ 100 سنة.")
else:
    # 5. إذا كان العمر أكبر من 100، نعرض رسالة خاصة
    print("أنت حكيم جدًا!")
```

## شرح الكود خطوة بخطوة:

### 1. طلب اسم المستخدم:
```python
name = input("من فضلك، أدخل اسمك الكامل (الاسم + اسم العائلة): ")
```
- `input()` هي دالة مدمجة في بايثون تستخدم لأخذ مدخلات من المستخدم على شكل نص.
- نطلب من المستخدم كتابة اسمه الكامل، ونخزنها في المتغير `name`.

### 2. طلب عمر المستخدم مع التحقق من صحته:
```python
while True:
    age_input = input("من فضلك، أدخل عمرك (رقم صحيح موجب): ")
    if age_input.isdigit():
        age = int(age_input)
        if age > 0:
            break
        else:
            print("الرجاء إدخال رقم موجب.")
    else:
        print("الرجاء إدخال رقم صحيح فقط.")
```
- نستخدم حلقة `while True` لكي نستمر في طلب الإدخال حتى نحصل على قيمة صحيحة.
- نستخدم `isdigit()` للتحقق مما إذا كانت جميع حروف السلسلة عبارة عن أرقام (أي أنه لا يحتوي على حروف أو رموز).
- إذا كانت القيمة رقمية، نحولها إلى عدد صحيح باستخدام `int()`.
- ثم نتحقق إذا كان العدد موجبًا (> 0).
- إذا لم تتحقق الشروط، نطبع رسالة خطأ ونعيد طلب الإدخال.

### 3. حساب السنوات المتبقية حتى 100 سنة وعرض رسالة الترحيب:
```python
if age < 100:
    years_left = 100 - age
    print(f"مرحبًا {name}! بقي لك {years_left} سنة حتى تبلغ 100 سنة.")
```
- إذا كان عمر المستخدم أقل من 100، نحسب الفرق بين 100 والعمر.
- نستخدم f-string (صياغة نصية تبدأ بـ f) لتسهيل تضمين المتغيرات داخل النص.
- نطبع رسالة ترحيب تحتوي على الاسم والسنوات المتبقية.

### 4. التعامل مع عمر أكثر من 100 سنة:
```python
else:
    print("أنت حكيم جدًا!")
```
- إذا كان عمر المستخدم أكبر أو يساوي 100، نطبع رسالة خاصة "أنت حكيم جدًا!".
