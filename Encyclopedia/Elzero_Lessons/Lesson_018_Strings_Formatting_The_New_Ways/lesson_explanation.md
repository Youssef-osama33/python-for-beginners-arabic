# درس بايثون رقم 18: تنسيق النصوص في بايثون – الطرق الحديثة (Strings Formatting The New Ways)

مرحبًا بك في درس جديد من دروس بايثون! في هذا الدرس سنتعرف على **تنسيق النصوص (Strings Formatting)** باستخدام الطرق الحديثة في بايثون. سنتعلم كيف نُنسق النص بطريقة سهلة ومرنة لجعل المخرجات أكثر وضوحًا وجمالًا.

---

## مقدمة: لماذا نحتاج إلى تنسيق النصوص؟

عندما نكتب برنامجًا، كثيرًا ما نحتاج إلى عرض نصوص تحتوي على متغيرات. مثلاً:

```python
name = "أحمد"
age = 25
print("اسمي " + name + " وعمري " + str(age) + " سنة.")
```

هذه الطريقة صحيحة لكنها ليست عملية أو مرنة، خصوصًا إذا كانت الجمل أطول أو المتغيرات كثيرة.

---

## الطرق القديمة لتنظيم النصوص

- استخدام **العلامة `%`** (طريقة قديمة لكنها لا تزال مستخدمة).
  
مثال:

```python
name = "أحمد"
age = 25
print("اسمي %s وعمري %d سنة." % (name, age))
```

لكن في هذا الدرس، سنتعرف على الطرق الحديثة التي تعتبر أفضل من حيث المرونة والوضوح.

---

## الطرق الحديثة لتنظيم النصوص في بايثون

### 1. استخدام `str.format()`

هذه الطريقة ظهرت في بايثون 3 وتعتبر أكثر مرونة.

#### الصيغة الأساسية:

```python
"نص يحتوي على {} و {}".format(value1, value2)
```

#### مثال:

```python
name = "أحمد"
age = 25
print("اسمي {} وعمري {} سنة.".format(name, age))
```

تقوم `{}` بتمثيل مكان سيتم استبداله بالقيم التي داخل `format()` حسب الترتيب.

---

#### استخدام الأرقام داخل الأقواس `{}` لتحديد ترتيب المعطيات

```python
print("اسمي {0} وعمري {1} سنة. نعم، {0}!")
```

- `{0}` تشير إلى أول قيمة في `format()`
- `{1}` تشير إلى القيمة الثانية

مثال:

```python
name = "أحمد"
age = 25
print("اسمي {0} وعمري {1} سنة. نعم، {0}!".format(name, age))
```

---

#### استخدام أسماء المتغيرات داخل `format()`

يمكن تمرير المتغيرات بالاسم:

```python
print("اسمي {name} وعمري {age} سنة.".format(name="أحمد", age=25))
```

أو مع متغيرات:

```python
name = "أحمد"
age = 25
print("اسمي {name} وعمري {age} سنة.".format(name=name, age=age))
```

---

#### تنسيق الأرقام

يمكننا تنسيق الأرقام بسهولة، مثلاً عرض عدد الأرقام بعد الفاصلة العشرية:

```python
pi = 3.1415926535
print("قيمة باي تقريباً: {:.2f}".format(pi))  # يعرض 3.14
```

- `.2f` تعني عرض رقم عشري بدقة منزلتين بعد الفاصلة.

---

### 2. استخدام الـ **f-strings** (من بايثون 3.6 وأحدث)

هي الطريقة الأكثر سهولة ووضوحًا.

#### الصيغة:

```python
f"نص يحتوي على {variable} مباشرة"
```

#### مثال:

```python
name = "أحمد"
age = 25
print(f"اسمي {name} وعمري {age} سنة.")
```

- هنا نضع حرف `f` قبل علامات التنصيص.
- داخل النص نضع المتغيرات بين `{}` وسيتم استبدالها بقيمها.

---

#### أمثلة أخرى على f-strings:

- تنسيق الأرقام:

```python
pi = 3.1415926535
print(f"قيمة باي تقريباً: {pi:.3f}")  # يعرض 3.142
```

- استخدام تعبيرات داخل `{}`:

```python
a = 5
b = 10
print(f"مجموع {a} و {b} هو {a + b}")
```

---

## ملخص الفرق بين الطريقتين

| النقطة               | `str.format()`                      | f-strings                         |
|----------------------|-----------------------------------|----------------------------------|
| سهولة الكتابة        | أقل                              | أسهل وأوضح                      |
| دعم التعبيرات داخل النص | لا يمكن بشكل مباشر                | يمكن استخدام
