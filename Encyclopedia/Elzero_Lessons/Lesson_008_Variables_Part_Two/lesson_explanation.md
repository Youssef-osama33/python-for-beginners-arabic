## مراجعة سريعة: ما هي المتغيرات؟

المتغير هو اسم يُعطى لمكان في الذاكرة لتخزين قيمة معينة يمكن تغييرها أثناء تنفيذ البرنامج.

مثال:
```python
age = 25
name = "Ahmed"
```

## 1. تغيير قيم المتغيرات

في بايثون يمكنك تغيير قيمة المتغير بسهولة. المتغير ليس ثابتًا، بل يمكن إعادة تعيينه لقيمة جديدة.

مثال:
```python
x = 10
print(x)  # النتيجة: 10

x = 20
print(x)  # النتيجة: 20
```

هنا قمنا أولًا بتعيين القيمة 10 للمتغير `x` ثم قمنا بتغييرها إلى 20.

---

## 2. أنواع البيانات الشائعة للمتغيرات

عند تعيين قيمة لمتغير، يمكن أن تكون القيمة من أنواع مختلفة:

- **الأعداد الصحيحة (int):** 1, 2, 100, -5
- **الأعداد العشرية (float):** 3.14, -0.5
- **النصوص (str):** "مرحبا", "Python"
- **القيم المنطقية (bool):** True أو False

مثال:
```python
age = 30           # int
height = 1.75      # float
name = "Sara"      # str
is_student = True  # bool
```

---

## 3. قواعد تسمية المتغيرات

عندما تسمي متغيرًا، يجب أن تتبع قواعد معينة:

- يجب أن تبدأ بحرف (a-z, A-Z) أو الشرطة السفلية `_`.
- يمكن أن يحتوي الاسم على أرقام ولكن ليس في البداية.
- لا تستخدم الكلمات المحجوزة في بايثون (مثل `if`, `for`, `while`).
- لا تستخدم مسافات، يمكنك استخدام الشرطة السفلية (_) للفصل بين الكلمات.

أمثلة صحيحة:
```python
my_var = 10
name1 = "Ali"
_age = 25
```

أمثلة خاطئة:
```python
2name = "No"      # يبدأ برقم -> خطأ
my-var = 5        # يحتوي على شرطة -> خطأ
for = 3           # كلمة محجوزة -> خطأ
```

---

## 4. تعيين قيم متعددة في سطر واحد

يمكنك تعيين أكثر من متغير في نفس السطر بسهولة:

```python
a, b, c = 5, 10, 15
print(a)  # 5
print(b)  # 10
print(c)  # 15
```

أيضًا يمكن تعيين نفس القيمة لعدة متغيرات:

```python
x = y = z = 100
print(x, y, z)  # 100 100 100
```

---

## 5. فهم الفرق بين المتغيرات والقيم

- **القيمة** هي البيانات نفسها (مثل الرقم 10 أو النص "مرحبا").
- **المتغير** هو اسم يمثل مكانًا في الذاكرة يحتوي على قيمة.

مثال:

```python
x = 10  # القيمة 10 مخزنة في المتغير x
y = x   # الآن y تحمل نفس القيمة 10
```
