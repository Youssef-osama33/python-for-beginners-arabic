# 🛡️ Cyber Security with Python

Welcome to the Professional Cyber Security track! This section is designed for security researchers, ethical hackers, and developers who want to build secure systems.

## 🗺️ Learning Path
1.  **[Ethical Hacking](./Ethical_Hacking/):** Network scanning, vulnerability assessment, and exploitation.
2.  **[Cryptography](./Cryptography/):** Encryption, decryption, hashing, and digital signatures.
3.  **[Network Security](./Network_Security/):** Packet sniffing, firewall bypass, and secure communication.
4.  **[Malware Analysis](./Malware_Analysis/):** Reverse engineering and behavior analysis.
5.  **[Web Security](./Web_Security/):** SQL injection, XSS, and CSRF prevention.

---

## 🛠️ Essential Libraries
- **Scapy:** Packet manipulation and network analysis.
- **Cryptography:** Secure encryption and decryption.
- **Requests:** HTTP library for web security testing.
- **Paramiko:** SSHv2 protocol implementation.
- **BeautifulSoup:** Web scraping for vulnerability scanning.
