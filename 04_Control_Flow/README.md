# 04_Control_Flow (التحكم في التدفق) 🐍

في هذا الدرس سنتعرف على الشروط (If) والحلقات (Loops) في بايثون.

## ⚖️ الشروط (If Statements)
تستخدم لاتخاذ قرارات في الكود بناءً على شرط معين:
- `if` (إذا)
- `elif` (إذا كان)
- `else` (وإلا)

```python
age = 18
if age >= 18:
    print("أنت بالغ")
else:
    print("أنت قاصر")
```

## 🔄 الحلقات (Loops)
تستخدم لتكرار تنفيذ كود معين عدة مرات:
- `for` (لأجل)
- `while` (بينما)

```python
for i in range(5):
    print(i)
```
