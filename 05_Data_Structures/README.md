# 05_Data_Structures (هياكل البيانات) 🐍

في هذا الدرس سنتعرف على القوائم (Lists)، القواميس (Dictionaries)، والمجموعات (Sets) في بايثون.

## 📋 القوائم (Lists)
تستخدم لتخزين مجموعة من العناصر في متغير واحد:
- `list` (قائمة)
- `append()` (إضافة عنصر)
- `remove()` (حذف عنصر)

```python
fruits = ["تفاح", "موز", "برتقال"]
fruits.append("عنب")
print(fruits[0]) # تفاح
```

## 📖 القواميس (Dictionaries)
تستخدم لتخزين البيانات في شكل مفتاح وقيمة (Key-Value):
- `dict` (قاموس)
- `keys()` (المفاتيح)
- `values()` (القيم)

```python
person = {"name": "أحمد", "age": 25}
print(person["name"]) # أحمد
```

## 📦 المجموعات (Sets)
تستخدم لتخزين مجموعة من العناصر الفريدة (بدون تكرار):
- `set` (مجموعة)
- `add()` (إضافة عنصر)
- `remove()` (حذف عنصر)

```python
numbers = {1, 2, 3, 3, 4}
print(numbers) # {1, 2, 3, 4}
```
