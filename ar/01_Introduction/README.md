# 01_Introduction (المقدمة) 🐍

مرحباً بك في عالم بايثون! في هذا الدرس سنتعرف على ما هي بايثون وكيفية تثبيتها.

## ❓ ما هي بايثون؟
بايثون هي لغة برمجة عالية المستوى (High-level)، مفتوحة المصدر، وسهلة القراءة. تم إنشاؤها بواسطة **Guido van Rossum** في عام 1991.

## 🛠️ تثبيت بايثون
1. اذهب إلى [python.org/downloads](https://www.python.org/downloads/).
2. قم بتحميل النسخة المناسبة لنظام تشغيلك (Windows, macOS, Linux).
3. تأكد من تفعيل خيار **"Add Python to PATH"** أثناء التثبيت.

## 💻 أول كود لك (Hello World)
للتأكد من أن بايثون تعمل بشكل صحيح، افتح محرر الأكواد (مثل VS Code) أو الـ Terminal واكتب:

```python
print("مرحباً بك في عالم بايثون!")
```
