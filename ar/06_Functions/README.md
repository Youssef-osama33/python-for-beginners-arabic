# 06_Functions (الدوال) 🐍

في هذا الدرس سنتعرف على الدوال (Functions) في بايثون.

## 🛠️ الدوال (Functions)
الدالة هي عبارة عن كود برمجي يقوم بمهمة معينة، ويمكن استدعاؤه في أي وقت.

```python
def greet(name):
    print(f"مرحباً بك يا {name}!")

greet("أحمد")
```

## 📥 المدخلات (Arguments)
يمكن للدالة أن تأخذ مدخلات (Arguments) لتقوم بعمليات عليها.

```python
def add(x, y):
    return x + y

result = add(10, 5)
print(result) # 15
```

## 📤 المخرجات (Return)
تستخدم كلمة `return` لإرجاع قيمة من الدالة.
