# 07_Modules (الموديلات والمكتبات) 🐍

في هذا الدرس سنتعرف على الموديلات (Modules) والمكتبات (Libraries) في بايثون.

## 📦 الموديلات (Modules)
الموديل هو عبارة عن ملف بايثون يحتوي على دوال ومتغيرات جاهزة للاستخدام.

```python
import math
print(math.sqrt(16)) # 4.0
```

## 📚 المكتبات (Libraries)
المكتبة هي عبارة عن مجموعة من الموديلات الجاهزة التي تساعدك في القيام بمهام معينة.

```python
import random
print(random.randint(1, 10)) # رقم عشوائي بين 1 و 10
```

## 📥 استيراد الموديلات (Importing)
يمكنك استيراد الموديلات بعدة طرق:
- `import math` (استيراد الموديل بالكامل)
- `from math import sqrt` (استيراد دالة معينة من الموديل)
- `import math as m` (استيراد الموديل باسم مستعار)
