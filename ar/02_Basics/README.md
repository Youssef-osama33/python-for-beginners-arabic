# 02_Basics (الأساسيات) 🐍

في هذا الدرس سنتعرف على المتغيرات (Variables) وأنواع البيانات (Data Types) في بايثون.

## 📦 المتغيرات (Variables)
المتغير هو عبارة عن حاوية لتخزين البيانات في الذاكرة. في بايثون، لا تحتاج إلى تحديد نوع المتغير مسبقاً.

```python
name = "أحمد"
age = 25
is_student = True
```

## 📊 أنواع البيانات (Data Types)
بايثون تدعم عدة أنواع من البيانات:
- **int:** الأعداد الصحيحة (مثل 10, -5).
- **float:** الأعداد العشرية (مثل 3.14, 2.5).
- **str:** النصوص (مثل "مرحباً", 'بايثون').
- **bool:** القيم المنطقية (True أو False).
