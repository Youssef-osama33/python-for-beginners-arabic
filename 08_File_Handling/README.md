# 08_File_Handling (التعامل مع الملفات) 🐍

في هذا الدرس سنتعرف على كيفية قراءة وكتابة الملفات في بايثون.

## 📂 التعامل مع الملفات (File Handling)
تستخدم دالة `open()` لفتح الملفات، ويمكنك تحديد الوضع (Mode) الذي تريد استخدامه:
- `r` (قراءة)
- `w` (كتابة)
- `a` (إضافة)
- `x` (إنشاء)

```python
with open("test.txt", "w") as f:
    f.write("مرحباً بك في عالم بايثون!")
```

## 📖 قراءة الملفات (Reading)
تستخدم دالة `read()` لقراءة محتوى الملف.

```python
with open("test.txt", "r") as f:
    content = f.read()
    print(content)
```

## ✍️ كتابة الملفات (Writing)
تستخدم دالة `write()` لكتابة محتوى في الملف.
